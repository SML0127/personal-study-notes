## 클린 코드(Clean code)?        
클린 코드란 아래의 특징을 가진다.
1) 좋은 가독성을 가지며 
2) 수정(change)하기가 쉬워야 하며 
3) 두개 이상의 일을 하지 않으며 
4) 코드를 보면 코드를 작성한 사람이 코드를 케어(care)하고 누가 읽고/유지하는지 알게되며 (It shows the person who wrote the code really ‘Cares’ for the code and who is reading/maintaining it.)
